# Medical_Chatbot
Medical Chatbot with LLMs, LangChain, Pinecone, Flask &amp; AWS

## How to run?

### Step 01 - Clone the repository
```bash
git clone https://github.com/AsenDulakshanaKavinda/Medical_Chatbot.git
```

### Step 02 - Install UV package manager(install python and pip if you don't have)
```bash
pip install uv
```

### Step 03 - Create a virtual environment
```bash
uv venv .venv
```

### Step 04 - Create a virtual environment(.venv)
```bash
uv venv .venv
```

### Step 05 - Activate the virtual environment(.venv)
#### Windows
```bash
.venv/Scripts/activate
```
#### Linux / Mac os
```bash
source .venv/Scripts/activate
```

### Step 06 - Install all the dependencies to the virtual environment(.venv)
```bash
uv sync
```
